To run the software for the boxtask, curiosity, delusions and need for closure questionnaires:
Install PsychoPy 3 or later from https://www.psychopy.org/download.html
Open PsychoPy.
From PsychoPy, open all the files with a .py extension in thise directory.  They are:
_init.py
BoxTask.py
Constants.py
Form.py
InfoScene.py
Scale.py
Trial.py
Run BoxTask.py.  It takes a few seconds for the window to appear where participants enter ID, age and sex.
If you click elsewhere while you wait, it can happen that this window appears behind the PsychoPy window.
